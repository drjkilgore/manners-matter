Live Coach — preview

A talking-face etiquette coach powered by LiveAvatar. This preview uses the LiveAvatar embed (FULL mode): HeyGen's model answers, steered by the "context" you configure, with HeyGen's moderation.

Read before enabling for children. This preview does not run the app's coded child-safety guardrails (the never-keep-secrets / talk-to-a-trusted-adult logic). For a kid-facing launch, configure a strict, age-appropriate context/persona in the LiveAvatar dashboard — or use the guardrail-driven build. Keep Sandbox mode on while testing: it's free (no credits). Live sessions consume credits.
Set up

Get your LiveAvatar API key at app.liveavatar.com/developers (separate from your HeyGen key) and set it in Netlify as LIVEAVATAR_API_KEY. Create a kid-appropriate avatar + context in the LiveAvatar dashboard and paste their IDs below. The pre-filled IDs are HeyGen's public sandbox demo, so you can see it work first.
